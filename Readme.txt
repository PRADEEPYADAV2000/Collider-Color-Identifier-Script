Collider Color Identifier Script.
This repository provides a C# script that allows users to easily colorize colliders in Unity. The script enables selecting custom colors for colliders directly from the Inspector, making it highly useful for projects with multiple colliders for identification purposes.

===============================================================================================================================================
Features:
Custom Color Selection: Choose colors for colliders directly from the Unity Inspector.
Supports All Collider Types: Works with any collider attached to a GameObject.
Improved Scene Management: Ideal for debugging and organizing projects with multiple colliders.


===============================================================================================================================================
Usage:
Attach the script to any GameObject with colliders.
Choose the desired color from the Inspector for easy visualization and identification.

===============================================================================================================================================
